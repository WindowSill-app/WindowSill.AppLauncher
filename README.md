# WindowSill.AppLauncher

This repository contains the source code for the App Launcher extension for WindowSill.

# Build, Run & Debug

To build and run the AppLauncher extension, follow these steps:
1. Clone the repository.
1. Open the solution `WindowSill.AppLauncher.slnx` with Visual Studio, Visual Studio Code or Rider.
1. Start Debugging in your IDE of choice.
